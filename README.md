# Web-Servo
_______________________________________________________________
This Is Simple Web Server Server For Termux

---How To Install---

1. pkg install git
2. pkg install python
3. git clone https://github.com/Toxic-Omega/Web-Servo
4. cd Setup
5. python Setup.py
